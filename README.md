#Personal-Finance-Dashboard

A simple Excel project to analyze personal income, expenses, and savings using interactive dashboards.

##Objective

To track and analyze income and expenses using Excel and generate meaningful financial insights.

##Tools Used

- MS Excel
- Pivot Tables
- Pivot Charts
- Data Visualization

##Features

- Total Income, Expenses, and Savings summary
- Monthly income vs expenses analysis
- Category-wise expense breakdown
- Interactive slicers for filtering data

##Insights

- Household expenses contribute a major portion of total spending
- Highest spending category identified easily
- Savings calculated with a strong savings rate
- Salary is the primary source of income
- December recorded the highest expenditure

##Conclusion

This project demonstrates basic data analysis and dashboard creation skills using Excel.
